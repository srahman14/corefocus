## Key User Workflows

### Habit Formation

1. Create a new habit from the Habit Tracker
2. Log daily completions with one click
3. View streaks and heatmaps to stay motivated
4. Analyze patterns in the Analytics dashboard

### Goal Achievement

1. Set goals on the Goal Board with deadlines
2. Track milestones and progress
3. Reflect and adjust strategies based on analytics
4. Celebrate completed goals

### Focus & Productivity

1. Start a Pomodoro session in Focus Camp
2. Work during the focused interval
3. Take breaks between sessions
4. Review focus hours in the Analytics dashboard

### Personal Reflection

1. Write journal entries in Read Route
2. Organize with tags and topics
3. Search past entries for inspiration
4. Track journaling frequency
