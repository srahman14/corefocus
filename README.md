# corefocus
Habit tracking app with additional features such as journaling, time-blocking, heatmaps and more.

## Features

### Core Features
- **Habit Tracker** - Track and manage your daily habits with progress visualization
- **Focus Camp** - Pomodoro timer for focused work sessions and productivity management
- **Read Route** - Journaling feature to capture thoughts and track recent journal entries
- **Goal Board** - Visual representation of your goals with deadline-based organization
- **Analytics** - Dashboard analytics to track your progress and insights

### Additional Features
- Real-time sync across devices
- Dark mode support
- Responsive design for mobile and desktop
- User authentication with Firebase
- Persistent data storage with Firestore
